# 测试手写笔记清除功能

## 快速测试步骤

### 方法一：使用测试Activity（推荐）

1. **将图片复制到手机**：
   ```bash
   adb push D:\MyApplication3\c334229afeaf75c8387b2f29a8ef2ef.jpg /sdcard/Download/
   ```

2. **启动测试Activity**：
   ```bash
   adb shell am start -n com.gongkao.cuotifupan/.ui.TestHandwritingRemovalActivity -e image_path /sdcard/Download/c334229afeaf75c8387b2f29a8ef2ef.jpg
   ```

3. **在应用界面**：
   - 查看原图显示
   - 点击"开始处理"按钮
   - 等待处理完成（几秒钟）
   - 查看处理后的效果对比

### 方法二：在代码中直接测试

在MainActivity或其他地方添加测试代码：

```kotlin
import com.gongkao.cuotifupan.util.HandwritingRemovalTester

// 测试处理图片
lifecycleScope.launch {
    val imagePath = "/sdcard/Download/c334229afeaf75c8387b2f29a8ef2ef.jpg"
    val resultPath = withContext(Dispatchers.IO) {
        HandwritingRemovalTester.testProcess(imagePath, 0.6f)
    }
    if (resultPath != null) {
        Toast.makeText(this@MainActivity, "处理成功: $resultPath", Toast.LENGTH_LONG).show()
    }
}
```

## 方法二：在MainActivity中添加测试按钮

可以在MainActivity的菜单中添加一个测试选项，直接调用测试功能。

## 方法三：使用命令行工具（需要Android环境）

由于ImageEditor使用了Android API，需要在Android环境中运行。可以创建一个简单的测试工具类。

## 测试图片路径

- 本地路径：`D:\MyApplication3\c334229afeaf75c8387b2f29a8ef2ef.jpg`
- 手机路径：`/sdcard/Download/c334229afeaf75c8387b2f29a8ef2ef.jpg`

## 预期结果

处理后的图片应该：
- 手写笔记被清除或淡化
- 打印文字保留
- 背景保持相对完整

## 注意事项

1. 处理大图片可能需要几秒钟
2. 效果取决于图片质量和手写笔记的特征
3. 如果效果不理想，可以调整 `intensity` 参数（0.0-1.0）

## 1. 软件概述

### 1.1 软件简介

"公考错题复盘"是一款专为公务员考试（国考、省考）、事业单位招聘及各类职业资格考试备考人员研发的智能错题管理软件。本应用深度结合了OCR光学字符识别技术、图形图像处理技术与艾宾浩斯遗忘曲线记忆法，旨在解决考生在刷题过程中面临的"错题整理耗时"、"复习无计划"、"资料查找难"三大痛点。

### 1.2 核心设计理念

本软件区别于传统的拍照搜题软件，不提供现成的答案解析，而是专注于帮助用户**构建个人专属的错题知识库**。通过模拟真实的"纸笔"做题体验（手写标注功能）和科学的复习计划（间隔重复算法），强迫用户进行深度思考，从而真正掌握解题方法。

### 1.3 主要特点

- **智能识别**：采用先进的OCR技术，支持中英文混合识别，准确提取题目内容
- **自动检测**：自动扫描相册中的新题目图片，后台智能识别并加入题库
- **错题管理**：智能分类、标签管理、快速搜索，系统化管理错题
- **笔记卡片**：支持为题目添加笔记，制作复习卡片，随时随地复习
- **数学练习**：提供多种数学运算专项练习，提升计算能力
- **手写标注**：支持在图片上手写标注，模拟真实做题体验
- **悬浮截题**：支持悬浮窗快速截取题目，无需切换应用
- **离线使用**：核心功能支持离线使用，无需网络连接

### 1.4 适用人群

- 准备公务员考试的考生
- 需要整理错题的学习者
- 希望提高学习效率的学生
- 需要系统复习的备考人员
- 需要科学记忆方法的用户

## 2. 系统要求

### 2.1 硬件要求

- **操作系统**：Android 7.0 (Nougat) API Level 24 及以上版本
- **处理器**：建议八核处理器
- **内存**：建议 4GB RAM 及以上
- **存储空间**：至少 100MB 可用存储空间
- **相机**：建议后置 800 万像素摄像头（用于拍照识别）
- **屏幕**：支持触摸屏操作

### 2.2 软件要求

- Android 操作系统
- 支持相机和存储访问权限
- 支持悬浮窗权限（可选，用于悬浮窗功能）
- 支持通知权限（可选，用于接收通知）

### 2.3 权限说明

应用需要以下权限才能正常使用：

1. **相机权限**：用于拍摄实体试卷上的题目
2. **存储权限**：用于保存错题图片及数据备份，访问相册中的题目图片
3. **悬浮窗权限**：用于"悬浮截题"功能（可选）
4. **屏幕录制权限**：仅用于悬浮窗截题时获取屏幕截图（可选）
5. **通知权限**：用于接收扫描完成等通知（可选）

**隐私保护承诺**：
- 所有图片仅在您的设备本地处理
- 不会上传任何照片到服务器
- 您可以随时在系统设置中撤销权限

## 3. 安装说明

### 3.1 下载安装

#### 3.1.1 下载APK

**方法一：从应用商店下载**
1. 打开应用商店（如Google Play、应用宝、华为应用市场等）
2. 搜索"公考错题复盘"
3. 点击"安装"按钮
4. 等待下载完成

**方法二：从官方网站下载**
1. 访问官方网站
2. 找到下载页面
3. 下载APK安装包
4. 保存到手机存储

#### 3.1.2 安装APK

1. 打开下载的APK文件
2. 如系统提示"未知来源"，请在设置中允许安装未知来源应用
   - 打开"设置 → 安全"
   - 开启"允许安装未知来源应用"
   - 不同手机品牌的设置位置可能不同
3. 点击"安装"按钮
4. 等待安装完成
5. 安装完成后，点击"打开"启动应用

[图片：安装界面截图]

### 3.2 首次启动

#### 3.2.1 权限请求

首次启动应用时，系统会请求以下权限：

1. **相机权限**：
   - 用于拍照识别题目
   - 点击"允许"授予权限
   - 如拒绝，将无法使用拍照功能

2. **存储权限**：
   - 用于访问相册中的题目图片
   - 点击"允许"授予权限
   - 如拒绝，将无法使用相册导入功能

3. **通知权限**（可选）：
   - 用于接收扫描完成等通知
   - 建议授予，方便接收通知

4. **悬浮窗权限**（可选）：
   - 用于悬浮窗截题功能
   - 可在需要时再授予

[图片：权限请求界面截图]

#### 3.2.2 首次扫描

首次启动后，应用会自动扫描相册中的图片：

1. **扫描过程**：
   - 系统开始扫描相册中的图片
   - 显示扫描进度
   - 识别图片中的题目
   - 自动添加到题库

2. **扫描时间**：
   - 取决于相册中的图片数量
   - 默认扫描最近50张图片
   - 可在设置中调整扫描数量

3. **扫描完成**：
   - 显示扫描结果
   - 显示识别出的题目数量
   - 可以开始使用应用

**注意事项**：
- 首次扫描可能需要几分钟时间，请耐心等待
- 建议在WiFi环境下进行，避免消耗流量
- 扫描过程中可以继续使用应用的其他功能

- **笔记搜索**：搜索笔记内容

- **翻转复习模式**：卡片翻转查看答案

- **数学运算练习**：多种数学运算专项练习

  - 比重相关：比重计算、比例分析
- **练习历史记录**：查看历史练习记录

  - 颜色选择：多种颜色可选

## 9. 卡片模块操作

卡片模块是应用的核心复习功能之一，采用间隔重复算法（Spaced Repetition Algorithm）帮助您科学高效地复习错题。通过将题目制作成记忆卡片，您可以随时随地进行复习，加深记忆。

### 9.1 卡片模块入口

#### 9.1.1 进入卡片模块

**方法一：通过底部导航栏**
1. 打开应用主界面
2. 点击底部导航栏的"笔记和卡片"图标（第二个图标）
3. 应用切换到笔记和卡片模块
4. 在模块顶部可以看到"笔记"和"卡片"两个标签页
5. 点击"卡片"标签页，进入卡片列表

**方法二：从题目详情创建**
1. 在题目详情页面
2. 点击"添加笔记"按钮
3. 选择"创建记忆卡片"选项
4. 系统自动跳转到卡片编辑页面，并将题目内容填充到卡片中

[图片：卡片模块入口截图]

### 9.2 查看卡片列表

#### 9.2.1 卡片列表界面

进入卡片模块后，您将看到卡片列表界面，包含以下元素：

**顶部区域**：
- **搜索框**：位于页面顶部，支持按卡片内容搜索
- **排序按钮**：点击可切换排序方式（按更新时间/按创建时间/按标签）
- **筛选按钮**：可按标签筛选卡片

**中间区域**：
- **卡片列表**：以卡片形式展示所有记忆卡片
- **空状态提示**：如果还没有卡片，显示"还没有记忆卡片\n点击右上角+号创建"的提示

**底部区域**：
- **开始复习按钮**：显示待复习卡片数量，点击进入复习模式

#### 9.2.2 卡片信息展示

每个卡片卡片显示以下信息：

1. **卡片正面预览**：显示卡片正面的前几行文字
2. **卡片背面预览**：显示卡片背面的前几行文字（可展开查看）
3. **标签信息**：显示卡片关联的标签（如有）
4. **复习状态**：显示卡片的学习状态
   - **新卡片**：尚未开始学习的卡片
   - **学习中**：正在学习中的卡片
   - **待复习**：需要复习的卡片（显示下次复习时间）
   - **已掌握**：已熟练掌握的卡片
5. **创建时间**：显示卡片创建的时间
6. **关联题目**：如果卡片来自题目，显示"来自题目"标识，点击可跳转到原题目

#### 9.2.3 卡片操作

**点击卡片**：
- 点击卡片任意位置，进入卡片详情/编辑页面

**长按卡片**：
- 长按卡片，弹出操作菜单
- 可选择"编辑"、"删除"、"复制"等操作

**滑动操作**：
- 左右滑动卡片，可快速删除或标记

[图片：卡片列表界面截图]

### 9.3 创建记忆卡片

#### 9.3.1 创建新卡片

**步骤一：进入创建页面**
1. 在卡片列表页面，点击右上角的"+"按钮
2. 或点击空状态提示中的"创建"按钮
3. 系统跳转到卡片编辑页面

**步骤二：输入卡片正面内容**
1. 在"提示"输入框中输入卡片正面内容
   - 正面通常是题目、问题或关键词
   - 例如："什么是马克思主义？"
   - 支持多行输入，可输入较长的内容
2. 如需添加图片，点击"添加正面图片"按钮
   - 选择图片来源（拍照或从相册选择）
   - 图片将显示在卡片正面

**步骤三：输入卡片背面内容**
1. 在"内容"输入框中输入卡片背面内容
   - 背面通常是答案、解析或详细说明
   - 例如："马克思主义是关于自然、社会和人类思维发展一般规律的科学"
   - 支持多行输入，可输入详细的解析内容
2. 如需添加图片，点击"添加背面图片"按钮
   - 选择图片来源（拍照或从相册选择）
   - 图片将显示在卡片背面

**步骤四：设置标签（可选）**
1. 在"标签"输入框中输入标签
   - 多个标签用逗号分隔，如："行测,数量关系,基础"
   - 标签用于分类和管理卡片
   - 可以点击标签输入框右侧的"选择标签"按钮，从已有标签中选择

**步骤五：选择卡包（可选）**
1. 点击"卡包"选择器
2. 从卡包列表中选择一个卡包
   - 如果还没有卡包，系统会使用"默认卡包"
   - 可以创建新卡包来组织卡片

**步骤六：保存卡片**
1. 检查卡片内容是否完整
2. 点击右上角的"保存"按钮（对号图标）
3. 系统保存卡片并返回卡片列表
4. 新创建的卡片显示在列表顶部

[图片：创建卡片界面截图]

#### 9.5.1 开始复习

#### 9.5.2 复习界面介绍

进入复习模式后，您将看到复习界面：

**顶部区域**：
- **进度显示**：显示当前复习进度，如"3/10"
- **统计信息**：显示"新: 2 | 学习: 3 | 复习: 5"
  - **新**：尚未开始学习的新卡片数量
  - **学习**：正在学习中的卡片数量
  - **复习**：需要复习的卡片数量

**中间区域**：
- **卡片显示区域**：显示当前卡片
  - 默认显示卡片正面（题目）
  - 点击卡片或"翻转"按钮可查看背面（答案）
  - 支持双指缩放查看细节

**底部区域**：
- **难度按钮**：四个按钮，用于标记掌握程度
  - **再次**：完全不记得，需要重新学习
  - **困难**：有点困难，需要多复习
  - **良好**：基本掌握，按正常间隔复习
  - **简单**：完全掌握，延长复习间隔

#### 9.5.3 复习流程

**第一步：查看卡片正面**
1. 系统显示卡片正面内容（题目）
2. 仔细阅读题目内容
3. 思考答案或解题思路

**第二步：查看卡片背面**
1. 点击卡片或"翻转"按钮
2. 系统显示卡片背面内容（答案）
3. 对比自己的答案与正确答案
4. 理解解析内容

**第三步：标记掌握程度**
1. 根据掌握情况，点击相应的难度按钮：
   - **再次**：如果完全不记得或答错了
   - **困难**：如果记得但不熟练
   - **良好**：如果基本掌握
   - **简单**：如果完全掌握
2. 系统根据您的选择，使用间隔重复算法计算下次复习时间
3. 自动切换到下一张卡片

**第四步：继续复习**
1. 重复上述步骤，继续复习下一张卡片
2. 可以左右滑动切换卡片
3. 复习过程中可以随时退出

#### 9.5.4 间隔重复算法说明

应用采用科学的间隔重复算法（类似 Anki），根据您的掌握程度自动安排复习时间：

**算法原理**：
- **再次**：卡片会立即重新出现，需要再次学习
- **困难**：较短间隔后复习（如：1天后）
- **良好**：正常间隔后复习（如：3天后）
- **简单**：较长间隔后复习（如：7天后）

**复习间隔递增**：
- 每次选择"良好"或"简单"，下次复习间隔会逐渐延长
- 例如：3天 → 7天 → 14天 → 30天 → 60天
- 如果选择"再次"或"困难"，间隔会缩短或重置

**智能调度**：
- 系统会优先显示需要复习的卡片
- 新卡片会在适当时候插入复习
- 确保复习效率最大化

### 9.10 卡片复习技巧

#### 9.10.1 创建优质卡片

**正面内容建议**：
- 使用简洁的问题或关键词
- 避免过长的题目
- 一个问题对应一个知识点

**背面内容建议**：
- 提供完整的答案或解析
- 可以包含解题步骤
- 可以添加记忆技巧或口诀

**标签使用建议**：
- 使用有意义的标签名称
- 按科目、题型、难度等维度分类
- 不要创建过多标签，保持简洁

#### 9.10.2 复习频率建议

**新卡片**：
- 建议每天学习5-10张新卡片
- 不要一次性学习太多，避免负担过重

**复习安排**：
- 每天复习待复习的卡片
- 系统会根据算法自动安排复习时间
- 坚持每天复习，效果最佳

**复习时间**：
- 建议利用碎片时间复习
- 每次复习10-20分钟
- 可以随时中断，下次继续

#### 9.10.3 掌握程度判断

**再次**：
- 完全不记得答案
- 答错了
- 需要重新学习

**困难**：
- 记得但不熟练
- 需要思考较长时间
- 答案不完整

**良好**：
- 基本掌握
- 能正确回答
- 但需要巩固

**简单**：
- 完全掌握
- 能快速准确回答
- 可以延长复习间隔

#### 9.10.4 复习效果提升

**主动回忆**：
- 看到正面时，先尝试回忆答案
- 不要直接看背面
- 主动回忆能加深记忆

**理解为主**：
- 不要死记硬背
- 理解答案的逻辑和原理
- 建立知识之间的联系

**定期回顾**：
- 定期回顾已掌握的卡片
- 确保长期记忆
- 防止遗忘

**结合题目**：
- 如果卡片来自题目，可以结合原题目复习
- 理解题目的解题思路
- 掌握相关知识点

3. 点击右侧的开关按钮，将开关切换到"开启"状态
4. 系统会请求存储权限（如果尚未授予）
5. 开启后，应用开始自动监听相册

**关闭自动监听**：
1. 进入设置页面
2. 找到"自动监听"选项
3. 点击右侧的开关按钮，将开关切换到"关闭"状态
4. 关闭后，应用停止自动监听

**注意事项**：
- 开启自动监听需要授予存储权限
- 自动监听会消耗一定的电量，建议在充电时使用
- 如果相册中有大量图片，首次扫描可能需要较长时间

#### 10.2.3 监听状态查看

**在主界面查看**：
- 在题目列表页面顶部，可以看到监听状态提示

- 扫描完成后，会显示"扫描完成，发现 X 道题目"的通知

### 10.3 扫描数量设置

#### 10.3.1 功能说明

扫描数量设置用于控制首次启动应用时，扫描相册中历史图片的数量。

**设置范围**：
- 最小值：10张
- 最大值：200张
- 默认值：50张
- 推荐值：50-100张

#### 10.3.2 设置扫描数量

**步骤一：进入设置**
1. 进入设置页面
2. 找到"扫描数量"选项

**步骤二：调整数量**
1. 点击"扫描数量"选项
2. 弹出数量选择对话框
3. 可以选择预设值（10、20、50、100、200）
4. 或手动输入自定义数量（10-200之间）
5. 点击"确定"保存设置

**步骤三：应用设置**
- 设置立即生效
- 下次启动应用时，会按照新设置的数量进行扫描
- 如果已经完成首次扫描，此设置对后续扫描无效

**设置建议**：
- **首次使用**：建议设置为50-100张，既能识别足够多的题目，又不会耗时过长
- **相册图片较少**：可以设置为全部扫描（200张）
- **相册图片很多**：建议设置为50张，避免首次启动时间过长

### 10.4 识别灵敏度设置

#### 10.4.1 功能说明

识别灵敏度用于控制题目识别的严格程度，影响哪些图片会被识别为题目。

**灵敏度等级**：
- **高灵敏度（严格）**：只识别非常明显的题目图片，准确率高但可能漏掉一些题目
- **中灵敏度（平衡）**：平衡准确率和识别率，推荐使用
- **低灵敏度（宽松）**：识别更多图片，但可能包含一些非题目图片

#### 10.4.2 调整识别灵敏度

**步骤一：进入设置**
1. 进入设置页面
2. 找到"识别灵敏度"选项

**步骤二：选择灵敏度**
1. 点击"识别灵敏度"选项
2. 弹出灵敏度选择对话框
3. 可以选择：
   - **高（严格）**：只识别明显的题目
   - **中（平衡）**：平衡识别率和准确率（推荐）
   - **低（宽松）**：识别更多图片
4. 点击"确定"保存设置

**步骤三：应用设置**
- 设置立即生效
- 后续扫描和识别会使用新的灵敏度设置
- 已识别的题目不受影响

**调整建议**：
- **默认使用中灵敏度**：适合大多数用户
- **如果误识别太多**：提高灵敏度（选择"高"）
- **如果漏掉很多题目**：降低灵敏度（选择"低"）
- **根据实际使用情况调整**：可以尝试不同设置，找到最适合的灵敏度

## 11. 常见问题

本章节整理了用户在使用过程中经常遇到的问题及解决方案，帮助您快速解决使用中的疑问。

### 11.1 安装与启动问题

#### Q1：应用无法安装怎么办？

**问题描述**：下载APK后无法安装，提示"未知来源"或"安装失败"。

**解决方案**：
1. **允许未知来源安装**：
   - 打开手机"设置"
   - 找到"安全"或"应用管理"
   - 开启"允许安装未知来源应用"或"允许从此来源安装"
   - 不同手机品牌的设置位置可能不同

2. **检查存储空间**：
   - 确保手机有足够的存储空间（至少100MB）
   - 清理不需要的文件和应用

3. **检查APK完整性**：
   - 重新下载APK文件
   - 确保下载完整，没有中断

4. **检查Android版本**：
   - 确保手机Android版本为7.0（API 24）及以上
   - 在"设置 → 关于手机"中查看Android版本

#### Q2：首次启动很慢怎么办？

**问题描述**：首次启动应用时，需要等待很长时间才能进入主界面。

**解决方案**：
1. **这是正常现象**：
   - 首次启动需要初始化数据库和扫描相册
   - 如果相册图片较多，可能需要几分钟时间
   - 请耐心等待，不要强制关闭应用

2. **减少扫描数量**：
   - 在设置中减少"扫描数量"（如设置为50张）
   - 可以加快首次启动速度

3. **关闭自动监听**：
   - 首次启动后，可以暂时关闭自动监听
   - 需要时再手动导入题目

#### Q3：应用闪退怎么办？

**问题描述**：打开应用后立即关闭，无法正常使用。

**解决方案**：
1. **重启应用**：
   - 完全关闭应用（从最近任务中清除）
   - 重新打开应用

2. **重启手机**：
   - 重启手机可以解决临时性问题
   - 清除系统缓存

3. **检查存储空间**：
   - 确保有足够的存储空间
   - 清理不需要的文件

4. **重新安装**：
   - 卸载应用
   - 重新下载并安装最新版本

5. **联系技术支持**：
   - 如果问题持续存在，请联系技术支持
   - 提供错误信息和手机型号

### 11.2 权限相关问题

#### Q4：为什么需要这么多权限？

**问题描述**：应用请求多个权限，用户担心隐私问题。

**权限说明**：
- **相机权限**：用于拍照识别题目（必需）
- **存储权限**：用于访问相册中的题目图片（必需）
- **悬浮窗权限**：用于悬浮窗截题功能（可选）
- **通知权限**：用于接收扫描完成等通知（可选）

**隐私保护承诺**：
- 所有图片仅在您的设备本地处理
- 不会上传任何照片到服务器
- 不会收集您的个人信息
- 您可以随时在系统设置中撤销权限

#### Q5：如何授予/撤销权限？

**授予权限**：
1. 首次启动时，系统会弹出权限请求对话框
2. 点击"允许"授予权限
3. 如果拒绝了权限，可以在系统设置中手动授予：
   - 打开"设置 → 应用 → 公考错题复盘 → 权限"
   - 开启相应权限

**撤销权限**：
1. 打开"设置 → 应用 → 公考错题复盘 → 权限"
2. 关闭相应权限
3. 注意：撤销权限后，相关功能将无法使用

#### Q6：悬浮窗权限在哪里设置？

**问题描述**：找不到悬浮窗权限设置位置。

**解决方案**：
1. **Android 6.0及以上**：
   - 打开"设置 → 应用 → 公考错题复盘 → 权限"
   - 找到"显示在其他应用上层"或"悬浮窗"选项
   - 开启权限

2. **不同手机品牌**：
   - **小米**：设置 → 应用管理 → 权限 → 显示悬浮窗
   - **华为**：设置 → 应用 → 权限管理 → 悬浮窗
   - **OPPO/OnePlus**：设置 → 应用管理 → 应用权限 → 悬浮窗
   - **vivo**：设置 → 安全 → 应用权限 → 悬浮窗
   - **三星**：设置 → 应用 → 特殊访问权限 → 显示在其他应用上层

3. **从应用内跳转**：
   - 在Pro服务页面开启悬浮窗时
   - 应用会自动跳转到权限设置页面

### 11.3 识别相关问题

#### Q7：为什么识别不准确？

**问题描述**：OCR识别出的文字有错误，或识别不出题目。

**可能原因**：
1. **图片质量问题**：
   - 图片模糊、不清晰
   - 光线不足或过亮
   - 图片倾斜或变形

2. **题目区域问题**：
   - 题目被裁剪或遮挡
   - 题目区域不完整
   - 背景干扰过多

3. **识别设置问题**：
   - 识别灵敏度设置不当
   - 需要调整识别参数

**解决方案**：
1. **提高图片质量**：
   - 在光线充足的环境下拍照
   - 保持手机稳定，避免模糊
   - 确保题目完整显示在画面中

2. **优化图片编辑**：

   - 去除无关的背景内容
   - 调整图片亮度和对比度

3. **调整识别设置**：

   - 如果漏掉题目，降低识别灵敏度

4. **手动编辑**：
   - 识别不准确时，可以在题目详情页手动编辑
   - 修改识别出的文字内容

#### Q8：识别速度慢怎么办？

**问题描述**：识别一张题目需要很长时间。

**可能原因**：
1. 设备性能较低
2. 图片过大
3. 同时运行多个应用

**解决方案**：
1. **关闭其他应用**：
   - 关闭不必要的后台应用
   - 释放设备内存

2. **优化图片**：
   - 应用会自动压缩大图片
   - 可以手动裁剪图片，减少处理区域

3. **等待处理**：
   - 识别过程需要几秒钟，请耐心等待
   - 不要频繁点击或退出

4. **升级设备**：
   - 如果设备较旧，考虑升级设备

#### Q9：为什么有些图片识别不出题目？

**问题描述**：某些图片明明包含题目，但系统没有识别为题目。

**可能原因**：
1. **识别灵敏度设置过高**：
   - 灵敏度太高，只识别非常明显的题目
   - 可能漏掉一些题目

2. **图片特征不明显**：
   - 题目格式不标准
   - 缺少选项标记（A/B/C/D）
   - 文字过少或过多

3. **图片质量问题**：
   - 图片模糊或分辨率低
   - OCR无法识别文字

**解决方案**：
1. **降低识别灵敏度**：
   - 在设置中降低识别灵敏度
   - 重新扫描相册

2. **手动导入**：
   - 使用"手动导入"功能

3. **提高图片质量**：

**OCR**：光学字符识别（Optical Character Recognition），用于识别图片中的文字。

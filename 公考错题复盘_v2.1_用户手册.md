## 1. 软件概述
### 1.1 软件简介
"公考错题复盘"是一款专为公务员考试（国考、省考）、事业单位招聘及各类职业资格考试备考人员研发的智能错题管理软件。本应用深度结合了OCR光学字符识别技术、图形图像处理技术与艾宾浩斯遗忘曲线记忆法，旨在解决考生在刷题过程中面临的"错题整理耗时"、"复习无计划"、"资料查找难"三大痛点。

### 1.2 核心设计理念
本软件区别于传统的拍照搜题软件，不提供现成的答案解析，而是专注于帮助用户**构建个人专属的错题知识库**。通过模拟真实的"纸笔"做题体验（手写标注功能）和科学的复习计划（红黄绿灯复习法），强迫用户进行深度思考，从而真正掌握解题方法。

## 2. 系统详细要求
### 2.1 运行环境
*   **操作系统**：Android 7.0 (Nougat) API Level 24 及以上版本。
*   **硬件要求**：建议八核处理器，4GB RAM，后置800万像素摄像头。

### 2.2 权限详细说明
1.  **相机权限**：用于拍摄实体试卷。
2.  **存储权限**：用于保存错题图片及数据备份。
3.  **悬浮窗权限**：用于"悬浮搜题"功能。
4.  **屏幕录制权限**：仅用于悬浮窗截题时获取屏幕截图。

## 4. 详细操作指南

### 4.1 悬浮窗搜题使用步骤
本功能允许您在不离开第三方刷题应用（如粉笔、华图）的情况下，快速录入错题。

**步骤 1：开启功能**
1.  打开"公考错题复盘"APP，进入首页。
2.  点击底部导航栏的"设置"图标，进入设置页面。
3.  找到"悬浮窗搜题"选项，点击右侧的开关按钮。
4.  **系统反应**：APP会跳转至Android系统的"显示在其他应用上层"权限设置页面。
5.  在列表中找到"公考错题复盘"，点击进入，并将滑块开启。
6.  按手机"返回"键回到APP，此时开关显示为开启状态，屏幕右侧会出现一个半透明的"复"字圆形悬浮球。

**步骤 2：截取题目**
1.  按手机"Home"键回到桌面，此时悬浮球依然保留在屏幕上。
2.  打开第三方刷题APP，进入做题界面，找到您需要录入的错题。
3.  点击屏幕上的悬浮球。
4.  **系统反应**：屏幕稍微变暗，弹出"将开始截取屏幕内容"的系统安全提示。
5.  在提示框中点击"立即开始"（或"允许"）。

**步骤 3：框选与保存**
1.  **系统反应**：屏幕完全静止并变暗，进入框选模式。
2.  手指按住题目区域的左上角，向右下角拖动。
3.  **视觉反馈**：手指拖动区域会变亮，显示出您选中的题目内容。
4.  手指松开屏幕。
5.  **系统反应**：APP自动跳转至"图片裁剪与确认"页面，显示刚才截取的图片。
6.  （可选）拖动图片的四个角微调范围。
7.  点击右上角的"保存"按钮（对号图标）。
8.  **结果**：提示"保存成功"，题目已存入您的错题本。

### 4.2 错题拍摄与录入步骤
适用于纸质试卷或书本上的错题录入。

**步骤 1：启动相机**
1.  在APP首页，点击底部中间圆形的"+"号按钮。
2.  在弹出的菜单中，选择"拍照录入"。
3.  **系统反应**：启动相机预览界面。

**步骤 2：拍摄**
1.  将手机对准试卷上的错题，保持手机平稳。
2.  点击屏幕任意位置进行对焦，确保文字清晰。
3.  点击底部的圆形"拍摄"按钮。
4.  **系统反应**：画面定格，进入裁剪流程。

**步骤 3：裁剪与确认**
1.  屏幕上出现一个绿色边框的裁剪框。
2.  手指按住裁剪框的四个角或四条边，拖动以包裹住完整的题目区域（建议去除无关的页眉页脚）。
3.  点击底部右侧的"对号"按钮确认。
4.  **系统反应**：进入题目详情编辑页，系统后台自动开始OCR文字识别。
5.  点击右上角"完成"，题目正式入库。

### 4.3 图片编辑与手写标注步骤
模拟真实的纸笔做题体验，可在图片上进行圈画、演算。

**步骤 1：进入标注模式**
1.  在任何题目的详情页，点击图片右下角的"画笔"图标。
2.  **系统反应**：进入全屏图片编辑界面，底部出现工具栏。

**步骤 2：选择画笔与颜色**
1.  点击工具栏左侧的"画笔"图标（通常为铅笔形状）。
2.  点击工具栏中间的颜色圆点。
    *   点击 **红色圆点**：画笔变为红色，适合批改或标记错误。
    *   点击 **蓝色圆点**：画笔变为蓝色，适合书写演算过程。
    *   点击 **绿色圆点**：画笔变为绿色，适合标记正确答案。
    *   点击 **黑色圆点**：画笔变为黑色，适合补充题干缺失文字。

**步骤 3：书写与修改**
1.  单指在图片上滑动，即可画出线条或文字。
2.  若写错，点击工具栏的"撤销"图标（向左弯曲的箭头），撤销上一步笔画。
3.  或点击"橡皮擦"图标，然后在图片上擦除任意笔迹。

**步骤 4：保存标注**
1.  点击界面右上角的"保存"图标（软盘形状）。
2.  **系统反应**：屏幕下方弹出提示"标注已保存"。
3.  点击左上角返回键，回到题目详情页，此时您可以看到刚才的笔记已经叠加在原图上。

### 4.4 资料分析速算训练步骤
专为提升资料分析模块计算速度设计。

**步骤 1：选择训练项目**
1.  点击底部导航栏的"练习"可以进入练习中心。
2.  在列表中点击"三位数除法"（或您想练习的其他项目）。
3.  弹出设置弹窗，选择题目数量（如"10题"）。
4.  点击"开始练习"。

**步骤 2：答题（以除法为例）**
1.  屏幕中央显示算式，例如 `458 ÷ 125`。
2.  屏幕下方提示"误差范围 ±2%"。
3.  点击输入框，弹出数字键盘。
4.  在心中估算或草稿纸计算结果（例如约等于3.66）。
5.  在键盘上输入 `3.66`。
6.  点击键盘上的"确定"或屏幕上的"提交"按钮。

**步骤 3：反馈与下一题**
1.  **若回答正确**：输入框周围变为绿色，并短暂显示"正确"字样，自动跳转下一题。
2.  **若回答错误**：输入框周围变为红色，并显示正确答案（如"正确答案：3.664"），需手动点击"下一题"继续。

**步骤 4：查看报告**
1.  完成所有10道题后，系统跳转至"训练报告"页。
2.  查看本次训练的"正确率"（如 90%）和"平均耗时"（如 5.2秒）。
3.  根据评级（优秀/良好/合格）调整后续训练计划。

### 4.5 Pro会员激活步骤
**步骤 1：获取兑换码**
1.  您通过由于购买获得的12位或18位数字字母组合代码。

**步骤 2：输入兑换码**
1.  在APP首页，点击右上角的"..."菜单图标。
2.  选择"激活 Pro 服务"。
3.  在弹出的输入框中，输入您的兑换码。
    *   *注*：系统会自动将小写字母转换为大写，并自动添加连字符，您只需连续输入即可。

**步骤 3：验证与激活**
1.  点击"验证"按钮。
2.  **系统反应**：显示"验证中..."，随后显示"兑换码有效：包含365天Pro会员"。
3.  点击变亮的"立即激活"按钮。
4.  **结果**：弹出"激活成功"提示，页面显示您的会员到期时间。
